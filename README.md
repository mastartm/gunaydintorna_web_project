# Günaydın Torna Web Sitesi Projesi

Bu proje, bir torna atölyesinin dijital dünyadaki varlığını oluşturmak, sunduğu hizmetleri tanıtmak ve potansiyel müşterilerle iletişim kurmak amacıyla geliştirilmiştir.

## 🚀 Kullanılan Teknolojiler

* **Frontend:** HTML5, CSS3, JavaScript (Sade ve hızlı bir kullanıcı deneyimi için).
* **Backend (Planlanan):** .NET Core API & Entity Framework (Veri yönetimi için).
* **Database (Planlanan):** SQL Server (İletişim formları ve ürün yönetimi için).

## 🛠️ Özellikler

- **Hizmet Tanıtımı:** Atölyede yapılan teknik işlemlerin detaylı sergilenmesi.
- **Dinamik İletişim Formu:** (Geliştirilme aşamasında) Müşterilerin isim, e-posta ve telefon numarası ile teklif isteyebileceği yapı.
- **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu arayüz.

## 📂 Proje Yapısı

- `index.html` -> Ana giriş sayfası.
- `/css` -> Stil dosyaları.
- `/js` -> Dinamik etkileşimler için scriptler.
- `/img` -> Makine ve parça görselleri.
